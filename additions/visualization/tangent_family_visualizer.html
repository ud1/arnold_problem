Подсказка
Для y = x²: ℓ(t): y = 2 t x - t² ℓ(a) ∩ ℓ(b) = ((a+b)/2, ab) При арифметической прогрессии tᵢ = t₀ + i h и фиксированной сумме i+j = s: x = t₀ + (s h)/2 = const => точки лежат на одной вертикали. При геометрической прогрессии tᵢ = t₀ qⁱ и фиксированной сумме i+j = s: y = t₀² qˢ = const => точки лежат на одной горизонтали.
